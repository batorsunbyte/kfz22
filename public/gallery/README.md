# Gallery Images

Echte Werkstatt-Fotos hier ablegen.

## So fügst du ein Bild hinzu

1. Bild in diesen Ordner kopieren — z.B. `werkstatt-1.jpg`
2. In `data/gallery.ts` den passenden Eintrag bearbeiten:
   - `placeholder: false` setzen
   - `src` auf den Dateinamen anpassen, z.B. `'/gallery/werkstatt-1.jpg'`
   - `alt` und `caption` anpassen, falls dein Foto was anderes zeigt

## Empfehlungen

- **Format:** JPG oder WebP
- **Größe:** max. 1920×1080 (16:9) oder 1600×1200 (4:3)
- **Dateigröße:** unter 300 KB pro Bild (sonst lädt die Seite langsam)
- **Benennung:** `werkstatt-N.jpg`, `arbeit-N.jpg`, `team-N.jpg` (lowercase, keine Umlaute, keine Leerzeichen)

## Optimierung

Wenn dein Bild zu groß ist, kannst du es online komprimieren:
- https://squoosh.app/ (kostenlos, läuft im Browser)
- https://tinypng.com/ (kostenlos für JPGs)
